## Описание проекта
Система для построения и анализа деревьев атак на информационную систему с использованием графового подхода.

## Функциональные возможности
- Построение дерева атак (графовое представление)
- Анализ метрик риска (вероятность успеха, стоимость реализации)
- Интерактивная визуализация графов
- Генерация отчётов с рекомендациями по митигации

## Технологический стек
- **Python 3.11+** — ядро анализа, графы (NetworkX), веб-интерфейс (FastAPI)
- **Go 1.21+** — высокопроизводительные расчёты метрик
- **Docker, Docker Compose** — контейнеризация
- **GitHub Actions** — CI/CD
